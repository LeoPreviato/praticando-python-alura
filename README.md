# 🐍 Praticando Python - Alura

Repositório criado para armazenar todos os projetos desenvolvidos durante a trilha **Desenvolvimento Back-End Python** no curso **Praticando Python:
Trabalhando com projetos** da Alura.

O objetivo deste repositório é registrar minha evolução na linguagem Python, colocando em prática os conceitos aprendidos por meio de pequenos projetos.

---

## 🎯 Objetivo

- Praticar Python através de projetos.
- Consolidar os conceitos aprendidos durante o curso da Alura.
- Desenvolver boas práticas de programação.
- Praticar Git e GitHub durante o desenvolvimento.
- Construir um portfólio de projetos para acompanhar minha evolução.

## 🚀 Como utilizar

Clone este repositório:

```bash
git clone git@github.com:LeoPreviato/praticando-python-alura.git
```

Depois acesse o projeto desejado:

```bash
cd projeto-01-calculadora-gorjeta
```

Execute normalmente com Python:

```bash
python main.py
```
