# Calculadora de Gorjeta

## Funcionalidades

- Calcula o valor da gorjeta.
- Calcula o valor total da conta.
- Permite escolher a porcentagem da gorjeta.
- Valida entradas inválidas.
- Impede valores negativos para a conta e para a gorjeta.
- Impede letras, caracteres e espaços em branco para conta e gorjeta

## Tecnologias

- Python 3

## Como executar

```bash
python main.py
