# 🪪 Validador de CPF



## 📖 Sobre o projeto

Este programa recebe um CPF informado pelo usuário e realiza uma validação básica, verificando:

- Se o CPF contém apenas números.
- Se possui exatamente 11 dígitos.

Ao final, o programa informa se o CPF possui um formato válido ou inválido.

> **Observação:** esta versão realiza apenas a validação do formato do CPF. A verificação dos dígitos verificadores poderá ser implementada em versões futuras.

---
