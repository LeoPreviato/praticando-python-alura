# Projeto 03 - Contador de Vogais

## 📖 Sobre

Este programa recebe um texto informado pelo usuário e conta quantas vogais (`a`, `e`, `i`, `o`, `u`) existem na frase, independentemente de serem maiúsculas ou minúsculas.

## 🚀 Funcionalidades

- Recebe um texto digitado pelo usuário.
- Converte o texto para letras minúsculas.
- Percorre cada caractere da string.
- Conta a quantidade de vogais.
- Exibe o total de vogais encontradas.
